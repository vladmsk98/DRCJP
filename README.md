# DRCJP Gallery

Галерея артов с интерактивным интерфейсом и поддержкой тёмной/светлой темы.

## Описание

Галерея собственных артов с названиями на японском языке и подробным описанием. Проект состоит из:
- Интерактивной веб-галереи (фронтенд на GitHub Pages)
- Поддержки тёмной/светлой темы
- Адаптивного дизайна

## Арты

Каждый арт содержит:
- Название на японском языке
- Дату создания
- Описание с переводом и значением
- Изображение

Примеры названий:
- SHIZUKESA (静けさ) - тишина, спокойствие
- HEION (平静) - уравновешенность, покой
- KIBUN TENKAN (気分転換) - смена настроения

## Технологии

- HTML5
- CSS3
- JavaScript (ES6+)
- Адаптивный дизайн
- Поддержка темы
