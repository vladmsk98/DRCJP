# DRCJP - Галерея артов

Галерея собственных артов с возможностью фильтрации, сортировки и деления.

## Особенности

- **Просмотр артов**: Просмотр коллекции собственных произведений искусства
- **Навигация**: Кнопки "Предыдущий" и "Следующий" для переключения между артами
- **Поиск**: Поиск по названию и описанию
- **Фильтрация**: Фильтрация по названию, описанию, избранному
- **Фильтр по тегам**: Выбор тегов для фильтрации артов
- **Сортировка**: Сортировка по названию (А-Я, Я-А) и ID (возрастание, убывание)
- **Теги**: Просмотр тегов для каждого арта
- **Информация о файле**: Просмотр формата и разрешения изображения
- **Избранное**: Добавление артов в избранное с сохранением в localStorage
- **Увеличение изображения**: Открытие изображения в увеличенном виде
- **Поделиться**: Деление арта с выбором платформ (ВК, Telegram, Одноклассники, Bluesky, копировать ссылку)
- **Темы**: Поддержка светлой и тёмной тем оформления
- **Адаптивность**: Поддержка различных размеров экранов

## Использование

### Навигация
- Используйте кнопки "Предыдущий" и "Следующий" для переключения между артами
- Используйте клавиши стрелок ← → для навигации
- Используйте клавиши Home/End для перехода к первому/последнему арту

### Поиск и фильтрация
- Введите текст в поле поиска для поиска по названию и описанию
- Выберите тип фильтрации из выпадающего списка
- Используйте фильтр по тегам, кликая на теги

### Сортировка
- Выберите тип сортировки из выпадающего списка

### Избранное
- Нажмите "Добавить в избранное" для добавления текущего арта
- Используйте фильтр "Только избранное" для просмотра избранных артов

### Поделиться
- Нажмите "Поделиться артом" для открытия меню деления
- Выберите платформу для деления

### Увеличение изображения
- Кликните на изображение для открытия в увеличенном виде

### Переключение темы
- Нажмите кнопку "Сменить тему" в правом верхнем углу

## Горячие клавиши

- `←` / `→`: Навигация между артами
- `Home`: Первый арт
- `End`: Последний арт
- `Esc`: Очистить фильтр / закрыть модальные окна
- `Ctrl + F`: Добавить/удалить текущий арт из избранного
